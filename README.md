# unsplash_demo
Вопрос: "Обосновать в письменном виде (в README) свой выбор архитектуры и библиотек
 которые использовались при разработке."

Ответ: Для построения приложения использовал "Чистую Архитектуру". Выбор обусловлен тем,
что при данном подходе легко добавлять и модифицировать функционал приложения просто меняя
слои, например добавить новое апи, функцию поиска или бесконечную ленту контента итд. 
Выбор сторонних бибилиотек обоснован скорее привычкой, в слое презентации для контроля 
стейта использовал Stacked(это удобная либа на основе Provider), для работы с http - Dio.
Считаю, эти библиотеки прекрасно справляются со своей задачей.
